# Global Banking Empowerment

Professional banking platform with Next.js + Supabase + Tailwind CSS

## Features
✅ User Authentication ✅ Dashboard ✅ Admin Portal ✅ Demo Data ✅ Notifications

## Tech Stack
- Next.js 14
- React 18
- Supabase (Auth + Database)
- Tailwind CSS
- TypeScript

## Setup

1. `npm install`
2. Create `.env.local` with Supabase credentials
3. `npm run dev`

## Deployment
Deploy to Vercel with GitHub integration
